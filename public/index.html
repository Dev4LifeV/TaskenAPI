Bem-vindo a API Tasken!

API desenvolvida para o desafio do processo seletivo referente a vaga de Desenvolvedor Node.js Júnior.
Esta API consiste em um cadastro de produtos baseada em CRUD (Create - Read - Update - Delete).
A API fora desenvolvida em ambiente Node.js utilizando o framework NestJS fazendo uso da linguagem TypeScript e utilizando um ORM, cujo seria o Sequelize.

GET /produtos

Descrição

Retorna todos os produtos do banco de dados

URL

https://tasken-api.herokuapp.com/produtos

Informações de recursos

Formato de resposta JSON
Necessário autenticação? Não

Status code

200 para sucessão do retorno
500 erro no servidor

Parâmetros

Não são necessários quaisquer parâmetros

POST /produtos

Descrição

Cadastra um novo produto no banco de dados

https://tasken-api.herokuapp.com/produtos

Informações de recursos

Formato de resposta JSON
Necessário autenticação? Não

Status code

201 para sucessão do retorno
500 erro no servidor
403 os dados enviados do produto são inválidos

Parâmetros

Não são necessários quaisquer parâmetros

Exemplo do body

{
"nomeProduto": "Nome do produto",
"fabricante": "Nome do fabricante",
"quantidade": 0,
"valor": 0.00
